## File Description

- `severity_judge.py`  
  The **original version** of the evaluation prompt, which did **not explicitly define severity score criteria**.

- `severity_judge2.py`  
  The **updated version** that applies the **new severity score definitions**.

- `analyze.py`  
  A script for **comparing the results** produced by `severity_judge.py` and `severity_judge2.py`.

- `utils.py`  
  Contains **all definitions and shared utility functions** used across the evaluation pipeline.
